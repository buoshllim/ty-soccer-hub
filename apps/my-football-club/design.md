# My Football Club (MFC) — 설계 문서

작성일: 2026-06-08

## 개요

축구 구단 매니저 게임. 예산으로 선수를 사고 팔고, 포메이션과 전술을 설정한다.

---

## 핵심 결정사항

| 항목 | 결정 |
|---|---|
| 예산 | €250M 고정 (플레이어가 정하지 않음) |
| 선수 데이터 | 실제 이름 + 직접 만든 능력치 (Soccer Cards 95명 재활용) |
| 능력치 | PAC / SHO / PAS / DRI / DEF / PHY (FIFA 스타일 6개) |
| 선수 가격 | 능력치 합산 기반 자동 계산 |
| 판매 가격 | 구매가의 80% |
| 스쿼드 크기 | 최대 20명 (선발 11 + 후보 9) |
| 저장 방식 | Supabase DB + PIN 코드 (genius-tys 프로젝트 내 테이블) |
| 디자인 | FIFA/FC 스타일 (형광색, 강렬한 느낌) |

---

## 페이지 구조

### 홈
- 구단 이름 표시
- 이적시장 버튼
- 팀관리 버튼

### 이적시장
- 예산 표시
- 선수 목록 (사기/팔기)
- 홈으로 돌아가기 버튼

### 팀관리
- 포메이션 선택 및 선수 배치
- 전술 설정
- 홈으로 돌아가기 버튼

---

## 포메이션

지원 포메이션 3개:
- 4-3-3
- 4-4-2
- 3-5-2

배치 방식: 포지션 클릭 → 선수 목록 뜨면 선택

---

## 전술

**공격 스타일** (1개 선택)
- 점유율
- 빠른 역습
- 패스 위주

**수비 스타일** (1개 선택)
- 높은 압박
- 낮은 수비 블록
- 맨투맨

---

## 선수 데이터

- Soccer Cards의 95명 선수 재활용
- 능력치: PAC / SHO / PAS / DRI / DEF / PHY 새로 부여
- 가격: 능력치 합산 기반 자동 계산
